# 🚀 Desafios de Lógica - JavaScript

Bem-vindo ao meu laboratório de algoritmos! Este repositório é o meu "campo de treinamento" onde aplico conceitos de lógica de programação, manipulação de arrays, estruturas condicionais e cálculos matemáticos utilizando **JavaScript**.

O objetivo aqui é manter uma base de código organizada, documentada e em constante evolução, praticando desde operações básicas até desafios de lógica mais complexos (frequentemente baseados em desafios de plataformas como HackerRank).

---

## 🛠 Como utilizar
1. Clone este repositório.
2. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
3. Navegue até a pasta `desafiosJS/`.
4. Execute qualquer arquivo com o comando: `node nome-do-arquivo.js`.
